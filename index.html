What is Chativate?

a free software project that aims to offer a secure chat app that cares about user's privacy.

What makes Chativate different?

Most popular chat apps are owned by corporations which means that basically they own everything the user has including chat history and personal data which invades privacy even if their intentions were good.


Chativate provides 2 versions:

Client/Server

Peer-To-Peer


Chativate is so different from any other chatting app, it doesn't own any of the user's data.
